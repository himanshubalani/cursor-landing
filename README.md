# Dev Tool Landing Page - Cursor

## Web Dev Cohort 2026

**Assignment:** Recreate a desktop-first developer tool landing page inspired by the Cursor website, focusing on visual and structural accuracy using only HTML and CSS.

---

## Project Overview

This project is a static landing page closely modeled after the [Cursor](https://www.cursor.so/) website. The goal was to match the original site's layout, typography, color palette, and overall visual hierarchy as accurately as possible, without adding interactivity or animations.

**Note:**
> I received a takedown request, so I cannot deploy this website publicly. Instead, a video walkthrough of the final output is provided below.

## Sections Implemented

1. **Top Navigation Bar**
   - Logo, navigation links, and primary CTA
   - Dark background
2. **Hero Section**
   - Main headline, description, CTAs
   - Large product screenshot/mockup
3. **Trusted By / Logos**
   - Row of company logos
4. **Feature Sections (4 blocks)**
   - Two-column layout (text + image)
   - Alternating image/text positions
5. **Feature Cards Section**
   - Section title and grid of 3 cards
6. **Testimonials**
   - Quote cards with name and role
7. **Changelog / Updates**
   - List of updates with date
8. **Team / About**
   - Large image, short description, CTA
9. **Final CTA**
   - Big heading and single button
10. **Footer**
    - Multi-column links and company info

## Fonts & Colors Used

- **Fonts:**
  - [Inter](https://fonts.google.com/specimen/Inter) (for body and UI text)
  - [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) (for code snippets)
- **Colors:**
  - Dark backgrounds: #18181B, #23272F
  - Accent: #6366F1, #7C3AED
  - Text: #FFFFFF, #A1A1AA, #18181B
  - Grays: #F4F4F5, #E4E4E7, #D4D4D8

All colors and font weights were chosen to closely match the Cursor brand and reference site.

---

## Submission Notes
- No JavaScript, TailwindCSS, or AI tools were used.
- No responsiveness or animations were implemented (desktop-only, static site).
- All structure and styling is hand-coded in HTML and CSS.
- Brand assets and icons were matched as closely as possible to the original Cursor site.
